# EduPlatform - Educational Website for Science Students

A complete educational platform with math tools, study materials, and AI-powered learning for Class 11-12 Science students.

## Features

- 10 Interactive Math Tools
- 5 AI-Powered Learning Features
- 60+ Study Material Chapters
- Mobile-Responsive Design
- 100% Free & Open Source

## Tech Stack

- Next.js 14
- React 18
- Tailwind CSS
- Math.js
- Plotly.js
- Framer Motion
